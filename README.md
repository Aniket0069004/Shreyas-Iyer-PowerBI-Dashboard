
# The Sarpanch Files — Shreyas Iyer Power BI Dashboard

A 5-page interactive Power BI report analysing Shreyas Iyer's batting career.

## Pages
- **Home** — overview and navigation
- **IPL Career Journey** — season-by-season runs (2015–2026) across DC, KKR, PBKS
- **International Masterclass** — ODI/T20I/Test stats, comparison vs Kohli, Rohit, SKY
- **Player Comparison** — aggression vs consistency scatter, chase averages
- **2026 Season Deep-Dive** — match-by-match breakdown, 279 runs @ SR 169

## Key Stats Tracked
- 4010 total IPL runs | Avg SR 121.49
- 2025 best season: 604 runs
- ODI average 47 (vs Kohli 59, Rohit 49)
- 2026: highest single-match SR of 209 vs SRH

## Tools
Power BI Desktop | DAX | Manual data collection
